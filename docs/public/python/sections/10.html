9. Numpy/Torch

9.1. Views

Conceptually. An ndarray is a metadata object pointing into a memory buffer (everything is a view). But view usually means “an array object sharing another array’s buffer.”

In PyTorch, a view is a tensor that shares the same underlying memory as another tensor but has different shape/strides/metadata.

Commands view-producing ops:

x.view(...)
x.reshape(...)      # sometimes view, sometimes copy
x.transpose(...)
x.unsqueeze(...)
x.expand(...)
