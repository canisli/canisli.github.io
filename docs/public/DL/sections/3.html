    • For a batch, we can run over each data point in parallel and average the results
    • We implement backpropagation for an MLP in pset 1.
    • We can view forward-backward as just a big neural network since the backpropagation step is determined by the intermediate hidden values which are determined by the weight matrices and activation functions. We call this a hypernetwork.
    • The backward network only consists of linear layers since the chain rule is always a product of Jacobian matrices.
    • We can optimize a single function or input with backprop.
