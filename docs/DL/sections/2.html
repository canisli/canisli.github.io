  • Representational power. Let number of layers count hidden and output layers.

    • 1 layer = linear decision surface
    • 2 layers (linear ReLU linear) is enough by UAT. Intuition is that an indicator function can be expressed as 4 ReLUs (Equation 12)
    • Interestingly, it is sometimes the case that deep nets require far fewer parameters to fit data than wide nets.

    • Why are neural networks a good architecture

      1. High capacity
      2. Differentiable
      3. Good inductive biases (e.g. CNN)
      4. Run efficiently on parallel hardware
      5. Build abstractions (layer by layer yields increasingly abstracted representations)
      • Apparently they are also the best models of computations in animal brains
    • Hebbian learning is an alternative to backpropagation.

  • Generalization theory

    • Deep nets lie in the double-descent regime (in practice we don’t see the U shape plus smooth right hand slope but just the latter).
    • Classical theory: more parameters = more complicated functions that overfit
    • Simplicity hypothesis: deep nets learn simple functions that generalize

  • Transfer Learning and Foundation Models

    • As the network trains on a supervised task like classification, deeper layers reorganize these features. By the final lyaers, the representations tend to cluster semantically (e.g. cats near dogs).
    • This motivates transfer learning: once a network has learned rich clustered representations, those embeddings can be reused for many tasks.
