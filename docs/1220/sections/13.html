13. Sublinear algorithms

13.1. Definition

  • Classical algorithms: read the input and compute answer
  • Online algorithms: provide answer after each read piece
  • Streaming algorithms: same as online but only remember a tiny amount of history

  • Sublinear algorithms: compute an answer in much less time than is needed to read the input.

    • We settle for “approximately correct” answers. There are two notions of this:

      1. Optimization: Return answer that is within of the correct one.
      2. Decision: Always return YES if the input is a YES instance with some false positives for inputs “close” to having the property.
