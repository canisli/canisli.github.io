3. Randomized Algorithms

  • Informally, a random algorithm is one whose output depends on input and random numbers

  • Two classes of random algorithms:

    • Monte Carlo = always fast and usually accurate, e.g. posterior sampling and computing pi.
    • Las Vegas = always correct and usually fast, e.g. rejection sampling
